What is SHA-256?

SHA-256 (Secure Hash Algorithm 256-bit) is a cryptographic hash function that produces a fixed-size 256-bit (32-byte) hash value. It is widely used in various applications, including data integrity verification, password storage, and digital signatures. The SHA-256 algorithm is part of the SHA-2 family, which is known for its strong security features and resistance to collision attacks.
