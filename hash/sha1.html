What is SHA-1 Hashing?

SHA-1 (Secure Hash Algorithm 1) is a cryptographic hash function that produces a 160-bit (20-byte) hash value, typically rendered as a 40-digit hexadecimal number. It is widely used in various applications, including data integrity verification, digital signatures, and password storage. Although SHA-1 is considered less secure than newer algorithms like SHA-256, it remains a popular choice for many legacy systems.
